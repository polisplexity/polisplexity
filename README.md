# Polisplexity Talks

Public GitHub Pages repository for narrated Polisplexity talks.

## Published Talks

- `elephant-cities-mice-companies/`
  - Spanish narration: `index.html`
  - English narration: `index-en.html`
  - Original Google Slides visuals are preserved as slide backgrounds.
  - Enriched scripts live in `speaker_script.md` and `speaker_script_en.md`.

## Editorial Rules

- Keep original slides as the visual source of truth.
- Add enrichment through narration notes, scripts, README files, and social packs.
- Treat historical talks as living archives: state the original date and avoid presenting them as new work.
- For Edgar Valdes personal LinkedIn posts, default to English.
- Telegram handoff is manual-first: send copy and assets for review/posting before automating social publication.

## Review Before Publishing Social Copy

- Confirm visible logos and partner references are acceptable.
- Confirm dates and historical framing are clear.
- Replace `PENDING_GITHUB_PAGES_URL` in social packs after the live URL is known.
- Send `telegram_publish_outbox.json` only after the public URL has been updated.
